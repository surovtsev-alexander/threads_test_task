# threads_test_task

----

## Description
В проекте два приложения:
- Модульные тесты `./Applications/UnitTests`.
- Необходимая программа `Applications/Logger`.

Приложения компилируются только в VisualStudio.
Для сборки каждого из приложений необходимо:
- в VisualStudio открыть соответствующую папку `./Applications/UnitTests` или `Applications/Logger`.
- открыть файл `CMakeLists.txt` и в нем нажать 'Ctrl + S'.
- можно собирать исполняемый файл.

Если у Вас не установлен 'visualgdb', то VisualStudio может не отображать весь список исходных кодов. Но сборка выполняться должна.
